#  SIMPLE LINUX TOOLKIT-MY FIRST A BIT ADVANCED SCRIPT AFTER LEARNING FOR FEW WEEKS

A full-screen terminal toolkit built in Bash with:
- arrow-key navigation
- system monitoring
- file automation
- plugin architecture
- audit logging system

---

## Features

### TUI Interface
- Arrow key navigation (↑ ↓)
- No typing numbers
- Full-screen CLI dashboard

### File Engine
- Auto file organization
- Duplicate detection (MD5 hashing)
- Category sorting

### System Dashboard
- CPU load
- memory usage
- disk usage
- top processes

### Plugin System
- Drop scripts into `/plugins`
- Auto-load at runtime
